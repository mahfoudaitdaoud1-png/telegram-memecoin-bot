## ☁️ STEP 6: DEPLOY TO CLOUD RUN (10 min)

### **6.1 Go to Cloud Run Console**

1. Go to https://console.cloud.google.com/run
2. Find your existing bot service
3. Click on it
4. Click **"Edit & Deploy New Revision"** (top)

### **6.2 Add Environment Variables**

1. Scroll down to **"Container"** section
2. Click **"Variables & Secrets"** tab
3. Click **"Add Variable"** (3 times for 3 new variables)

**Add these 3 NEW variables:**

**Variable 1:**
```
Name:  TRADING_WALLET_PRIVATE_KEY
Value: [paste your Phantom private key]
```

**Variable 2:**
```
Name:  SOLANA_RPC_URL
Value: [paste HTTP RPC URL from Helius]
```

**Variable 3:**
```
Name:  SOLANA_WS_URL
Value: [paste WebSocket URL from Helius]
```

### **6.3 Keep Existing Variables**

**Make sure these are still there:**
- TG (your Telegram token)
- ALERT_CHAT_ID
- Any other variables you had before

### **6.4 Deploy**

1. Scroll to bottom
2. Click **"Deploy"**
3. Wait 2-5 minutes for deployment

**You'll see:**
- "Creating Revision..." 
- "Routing traffic..."
- "✓ Service deployed"

**✅ CHECKPOINT:** Deployment shows "Ready" with green checkmark
